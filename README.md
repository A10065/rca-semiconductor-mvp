# Agentic RCA Smart Diagnostic Dashboard

這是一個專為半導體封裝測試 (OSAT) 設計的智慧化根因分析 (RCA) 儀表板。結合了 **Neuro-Symbolic AI** 推論與 **GraphRAG** 因果圖譜技術，提供專業級的失效診斷與 8D 報告生成功能。

## 🌟 核心功能

- **AI 邏輯診斷**：結合 LLM 的語意理解與本體論 (Ontology) 的物理驗證。
- **Galaxy GraphRAG**：基於 GraphRAG 風格的星系化因果網絡，視覺化知識路徑。
- **自動化 8D 報告**：遵循 ISO 品質標準，自動導出 D1-D8 的完整修正預防措施。
- **專業製造業 UI**：參考 Praxie RCA 介面，優化的明亮背景與模組化卡片設計。

## 🚀 快速開始

1. 克隆此倉庫：
   ```bash
   git clone https://github.com/USER/rca-semiconductor-mvp.git
   ```
2. 開啟 `dist/index.html` 即可運行（純前端展示版）。

## 🛠 技術棧

- **Frontend**: Vanilla HTML / CSS (Grid & Flexbox) / JavaScript
- **Visualization**: SVG-based Graph Logic
- **Architecture**: Neuro-symbolic RAG Framework
